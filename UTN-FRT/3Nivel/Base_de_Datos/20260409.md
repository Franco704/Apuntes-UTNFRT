# Gestión de Datos: Introducción y Lineamientos

##  Estándares de Diseño
- **Tablas:** Se definen utilizando **PascalCase** (ej. `UsuariosRegistrados`).
- **Atributos:** Por convención práctica, es preferible usar **snake_case** (ej. `fecha_nacimiento`).
- **Arquitectura:** Cada fragmentación de los discos representa una Base de Datos (BD).

## Hoja de Ruta de la Materia
- **Trabajos Prácticos:** Tendremos **5 TP's** en total.
- **Proyecto Integrador:** El último trabajo es un integrador donde se evaluarán bases de datos **No Relacionales (MongoDB)**. 
- **Flexibilidad:** Para el resto de los trabajos, el motor de BD es a elección.
- **Fechas y Modalidad:** - El último TP se defiende en clase el **10 de julio**.
    - Las clases prácticas probablemente sean virtuales.
    - Los grupos deben ser de **5 a 8 personas**.

> [!TIP] Consejos del Docente
> 1. **PostgreSQL:** Es el motor más potente. Destaca por su capacidad empresarial y su versatilidad para transformaciones No Relacionales y **Vectoriales**. Es el estándar para el salto a empresas.
> 2. **RAG (Retrieval-Augmented Generation):** Investigar sobre esta arquitectura de LLM para potenciar sistemas con datos propios.
> 3. **Anthropic Sonnet:** Recomendado para crear "ejércitos de chatbots" aplicados a la vida cotidiana (utilizando bases de datos vectoriales como PGVector en PostgreSQL).
> 4. **Bases de Datos Columnares:** PostgreSQL también puede manejarlas; son fundamentales para catálogos de venta a gran escala.
> 5. **Certificación MongoDB:** Aprovechar los **USD 50** de crédito que da la cuenta estudiantil para sacar la certificación oficial de forma gratuita.

## Mercado Laboral y Perspectivas
- **Especialización:** "Base de Datos" y "Seguridad" son los caminos más rápidos para monetizar conocimientos de forma independiente mientras se estudia.
- **Remuneración:** Un puesto de **DBA (Database Administrator)** ronda los **$3.8M netos** por mes.
- **Freelance:** Una BD bien documentada puede valorarse en el mercado internacional (ej. plataformas como Fiverr) en montos significativos (hasta **30k USD**).
