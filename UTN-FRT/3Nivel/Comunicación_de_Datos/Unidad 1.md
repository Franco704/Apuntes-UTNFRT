# 📶 Unidad 1: Sistemas de Comunicación

El objetivo principal de todo sistema de comunicaciones es intercambiar información entre dos o más entidades. Cuando nos comunicamos estamos compartiendo información, la cual puede ser local o remota. 

> [!QUOTE] Datos
> La palabra datos se refiere a hechos, conceptos e instrucciones presentados en cualquier formato acordado entre las partes que crean y utilizan dichos datos.

---

## 🌐 Redes de Telecomunicaciones
**¿Qué es una Red de Telecomunicaciones?**
- Es una infraestructura.
- Proporciona comunicación entre múltiples entidades.
- Lo hace de una manera eficiente.
- Usa distintas tecnologías (eléctricas, electrónicas, electromagnéticas, ópticas...).

Una red es un conjunto de dispositivos (a menudo denominados **nodos**) conectados mediante enlaces de un medio físico. Un nodo puede ser una computadora, una impresora o cualquier otro dispositivo capaz de enviar y/o recibir datos generados por otros nodos de la red.

### Transmisión de Datos
Es el intercambio de información entre dos dispositivos a través de alguna forma de medio de transmisión, como un cable de cobre, fibra óptica o electromagnético.
    
La efectividad del sistema depende de cuatro características principales:
1. **Entrega**
2. **Exactitud**
3. **Puntualidad**
4. **Retardo variable (jitter)**

---

## 🏗️ Esquema de un Sistema de Comunicación


Componentes del sistema:
* **Fuente:** Es la generadora de la información que se quiere enviar (audio, video, etc.). Esta señal puede ser analógica o digital. En el caso de que no sea eléctrica, se convierten en señales eléctricas a través de un transductor.
* **Transmisor:** Es el bloque encargado de transformar la señal que emana de la fuente para que pueda ser transportada de forma eficiente por el medio.
* **Canal:** Medio a través del cual se lleva la señal de información que se quiere transmitir.
* **Fuente de Ruido:** Conjunto de distorsiones o adiciones no deseados por la fuente de información que afectan a la señal.
* **Receptor:** Es el bloque encargado de transformar la señal (recibida).

## 📊 Criterios de Redes de Datos

* **Rendimiento:** * Depende de: el número de usuarios, el tipo de medio de transmisión, la capacidad del hardware y la eficiencia del software.
	* Se usan dos métricas principales: **Ancho de banda** y **Latencia**.
* **Fiabilidad:**
	* Se mide por: la frecuencia de fallo, el tiempo de recuperación de un enlace frente a un fallo, y la robustez de la red ante una catástrofe.
* **Seguridad:**
	* Incluyen protección de datos frente a accesos no autorizados, fallos, y modificación. 
	* También incluye la implementación de políticas, procedimientos y recuperación de interrupciones y pérdidas de datos.
